### 어쩌다 이런 짓을?
- 시중에 나와 있는 독서 메모 앱 중에 마음에 드는 게 없음
- 마침 Flutter 에 익숙해지기 위한 연습도 필요
- 실제로 마켓 배포까지 할지는 아직 잘? 모르겠음 이제 겨우 첫날일 뿐이라

### 만들고 싶은 기능
- 홈 화면
  - 그동안 메모한 책 목록이 가장 간단한 리스트 형태로 보임
  - 현재 읽고 있는 목록과 완독한 목록을 섹션 나눠서 표시
  - 읽고 있는 목록은 가나다순으로, 완독한 목록은 완독 날짜 + 가나다순으로 정렬
- 책 화면
  - 메모 목록이 보임. 목록을 어떻게 표시할지는 고민 필요
  - 읽기 시작한 날짜와 완독한 날짜를 설정할 수 있음
- 책의 메모 화면
  - 인용과 메모를 동시에 남길 수 있음
  - 인용 없이 메모만 남기거나 메모 없이 인용만 남기는 것도 가능
  - 인용은 숫자를 같이 남길 수 있음 (페이지, 화)
  - 인용과 메모는 둘 다 나중에도 수정 가능해야 함
  - 인용은 사진 형태일 수 있음
  - 마크다운 문법을 서포트할 수 있으면 좋음
- 모든 메모를 markdown 형태로 export 해서 다운받는 게 가능해야 함
- 아이폰 사진 앱의 extension 으로 새 메모 만들기 또는 기존 메모에 사진 추가하기가 가능해야 함
- 핸드폰 바탕화면에 위젯을 띄워놓을 수 있어야 함
- 기기 간 동기화
- 폰트 설정
- 검색
